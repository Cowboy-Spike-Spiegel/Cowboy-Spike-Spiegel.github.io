My first article
Created2022-01-19
​ 第一次尝试搭建个人博客还是在1月12日，第一次写随笔却在七日之后；很怠惰。 ​ 2022年又是不寻常的一年，开年即是冬奥会，导致笔者要在家呆上两个半月。准确来说假期开始于1月10日-概率论与数理基础考完。摆烂摆了十天。时至今日，终于静下心来。该干一些事情了。 ​ 那就想到哪写到哪，先随便整理一些吧。 学习目标 前端框架 CMU 的《深入理解计算机系统》、《计算机网络》 学习本校课程《计算机组成原理》《形式语言与自动机》 项目目标 Gaze 项目的开展与简单化实现 Java 通讯录的制作与验收 其他 再读《百年孤独》；阅读一些先生的作品（《朝花夕拾》优先） 布鲁斯口琴、绘画、足球 尽量不要648（寄！），月卡党万岁！ 先这样吧，刚入正轨，收心为上。
